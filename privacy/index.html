Privacy Policy

How tradecli handles Google Sign-In identity, product analytics, error events, and operational data.

Effective: June 1, 2026 Contact: contact@tradecli.in

Overview

tradecli is a terminal-based AI trading assistant for Indian markets. This policy explains what information we collect, what we avoid collecting, and how we use information to operate, secure, and improve the product.

For questions or privacy requests, contact contact@tradecli.in.

Information We Collect

When you sign in with Google or use tradecli, we may collect:

  • Google account identity information, including your email address, name, Google subject identifier, email verification status, and profile fields provided by Google.
  • Coarse request location and network metadata provided by Cloudflare, such as country or region, for security, abuse prevention, and aggregate analytics.
  • Product analytics events, including CLI launch, install or setup checks, login and logout events, persona selection, skill or tool category invocation, version, platform, architecture, install channel, and environment label such as development or production.
  • Error events needed to diagnose failures, such as error type, command area, status code, and non-sensitive runtime context.

Information We Do Not Collect in Analytics

tradecli analytics are designed to avoid sensitive trading and local-machine details. We do not intentionally collect the following in analytics events:

  • Prompts, responses, chat transcripts, model context, or generated trading analysis.
  • Broker credentials, API keys, OAuth tokens, session cookies, environment variable values, or other secrets.
  • Portfolio holdings, order details, positions, trade history, watchlists, account balances, or personally sensitive financial records.
  • Full tool arguments, tool results, browser page contents, local file paths, stack traces containing user data, or terminal command output.

How We Use Information

We use collected information to:

  • Authenticate users and link product analytics to signed-in accounts.
  • Operate, secure, debug, and improve tradecli.
  • Measure adoption, reliability, persona usage, skill usage, and setup friction.
  • Prevent abuse, spoofing, spam, and unauthorized access to admin analytics.
  • Respond to support, privacy, and compliance requests.

Google User Data

tradecli uses Google Sign-In to authenticate users and capture account identity. We request identity-related scopes such as OpenID, email, and profile. We do not request access to Gmail, Google Drive, Google Calendar, or other Google Workspace content.

We use Google user data only for authentication, account identity, account-linked analytics, security, and support. We do not sell Google user data.

tradecli's use and transfer of information received from Google APIs will adhere to the Google API Services User Data Policy, including the Limited Use requirements.

Sharing and Service Providers

We do not sell personal information. We may share information with service providers that help us run the product:

  • Google, for Google Sign-In authentication.
  • Cloudflare, for hosting, Workers, D1 database storage, request processing, security, rate limiting, and coarse request location metadata.
  • GitHub, for public website hosting, source distribution, and release infrastructure.

We may disclose information if required by law, to protect users, or to protect the integrity and security of tradecli.

Retention

We retain account identity, analytics, and error events for as long as reasonably needed to operate, secure, debug, and improve tradecli. We may aggregate or de-identify analytics for longer-term product measurement.

Security

We use HTTPS, Google Sign-In, signed access tokens, refresh-token rotation, server-side validation, Cloudflare rate limiting, and admin allowlists to protect tradecli systems. No system is perfect, but we design collection and access controls to reduce unnecessary exposure.

Your Choices

You can stop using tradecli or sign out of the CLI at any time. To request access, correction, or deletion of account-linked information, contact contact@tradecli.in.

Changes

We may update this policy as tradecli changes. Material changes will be reflected on this page with an updated effective date.

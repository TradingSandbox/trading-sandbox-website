tradecli: the agent-native trading OS for Indian markets

tradecli is the official site for the agent-native trading OS that runs local AI agents across brokers, charts, screeners, portfolios, and an AITradingOffice memory layer, so your market process becomes visible, queryable, and scalable.

Guardrails

Control stays visible at every step.

Agents can gather context, form recommendations, and prepare actions. Execution stays behind explicit approval, configuration, and an audit trail.

01

Observe

Read market, broker, chart, and office context.

02

Recommend

Explain thesis, risk, sizing, and invalidation.

03

Prepare

Draft the intended action without placing it.

04

Approve

Require operator review before execution.

05

Execute

Run only when the local config allows it.

06

Review

Write decisions, fills, and follow-up notes back.

Execution boundary

  • Broker credentials stay local to the machine.
  • Order placement is disabled unless explicitly configured.
  • Every recommendation and action path stays reviewable.
